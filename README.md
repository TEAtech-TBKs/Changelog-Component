# Changelog Component

A simple, responsive changelog/timeline component inspired by [roadmap.sh](https://roadmap.sh/projects/changelog-component).  
Displays project updates in a vertical timeline with dates, descriptions, and a call-to-action button.

## Live Project

You can see the live project here: [Changelog Component](https://teatech-tbks.github.io/Changelog-Component/)

## Project Page

Original project page: [Roadmap.sh Changelog Component](https://roadmap.sh/projects/changelog-component)

## Features

- Responsive vertical timeline
- Styled with pure CSS
- Includes a header and CTA button
- Works on desktop and mobile devices
